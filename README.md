Productivity Dashboard

A fully responsive productivity dashboard built using HTML, CSS, and JavaScript. This application combines multiple productivity tools into a single intuitive interface to help users manage their time, tasks, and daily goals efficiently.

---

🚀 Live Demo

https://prodashboardd.netlify.app/

---

✨ Features

- ⏰ Real-Time Clock
  Displays live time, date, and day dynamically.

- 🎨 Theme Switcher
  Allows users to change the overall theme and colors of the application.

- 🌤 Temperature Display
  Displays current temperature information.

- 📝 Task List (with Local Storage)
  Add, manage, and track tasks. Data is stored in the browser using local storage, so it remains saved after reload.

- 📅 Daily Planner
  Plan and organize a full 24-hour schedule.

- 💡 Motivational Quotes
  Displays motivational quotes to enhance user focus.

- ⏳ Pomodoro Timer
  Helps users stay productive using the Pomodoro technique.

- 🎯 Daily Goals
  Set and monitor daily goals.

- 📱 Fully Responsive Design
  Optimized for mobile, tablet, and desktop devices.

---

🛠 Tech Stack

- HTML
- CSS
- JavaScript (Local Storage API)
